# Expense Tracker

A simple ASP.NET Core MVC application to track daily expenses. Users can **add, edit, and delete expenses**, and view the **total expenses**.

## Features
- View a list of expenses with total calculation
- Add new expenses
- Edit existing expenses
- Delete expenses

## Technologies Used
- **ASP.NET Core MVC**
- **C#**
